# Portfolio Media Guidelines

Use these rules before adding files under `public/portfolio/`.

## Images

- Export gallery stills at realistic web display sizes.
- Prefer `webp` or compressed `jpg`/`png` over oversized originals.
- Keep a separate thumbnail file for homepage/card usage.
- Preserve the final displayed aspect ratio so layout shifts stay low.

## Videos

- Compress videos for web playback before adding them to the repo.
- Provide a poster image for every video.
- Prefer `mp4` or `webm`.
- Avoid using animated GIFs when an `mp4`/`webm` loop can replace them.

## Structure

- Put project thumbnails in `public/portfolio/industrial-design/thumbnails/`.
- Put per-project gallery media inside the matching project slug folder.
- Keep numbering/order explicit so gallery sequencing is stable in code.
